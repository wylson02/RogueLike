# RogueLike

Jeu roguelike console en C# (.NET) — projet original : machine à états, combat au tour
par tour, 4 niveaux scénarisés, sceaux, boss final.

## ⚔️ Les Sceaux de l'Abîme — Édition Deluxe

Le portage **jeu de bureau pixel art** (TypeScript + Canvas + Tauri, objectif Steam) se
trouve dans [`RogueLike-Deluxe/`](RogueLike-Deluxe/) :

- jouable immédiatement : ouvrir `RogueLike-Deluxe/dist/index.html` dans un navigateur ;
- fidèle 1:1 au gameplay C# (cartes, stats, scénario, boss phase 2) ;
- éclairage dynamique, particules, musique et bruitages, sauvegarde, manette, FR/EN ;
- projet Tauri 2 prêt pour produire les installeurs Windows/Mac/Linux.

Voir [`RogueLike-Deluxe/README.md`](RogueLike-Deluxe/README.md) pour le build et la
checklist Steam.

## Version console originale

Ouvrir `RogueLike/RogueLike.sln` avec Visual Studio / `dotnet run` dans `RogueLike/`.
